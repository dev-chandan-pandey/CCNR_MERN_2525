JSON.stringify() Object Example

This example demonstrates how to convert a JavaScript object into a JSON string using JSON.stringify(). The result will be logged to the console and displayed below. 


let student = {
  name: "Alice",
  age: 22,
  course: "Computer Science"
};

let jsonString = JSON.stringify(student);

// Expected Output (in console):
// {"name":"Alice","age":22,"course":"Computer Science"}
// (This is a string)
