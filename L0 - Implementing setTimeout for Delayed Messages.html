Asynchronous Code with setTimeout

This example demonstrates asynchronous code execution using setTimeout. "Message 1" and "Message 3" will appear immediately, followed by "Message 2 after 2 seconds" after a delay. Check your browser's developer console for the exact timing. 


function runAsynchronousTasks() {
  console.log("Message 1");

  setTimeout(() => {
    console.log("Message 2 after 2 seconds");
  }, 2000); // 2000 milliseconds = 2 seconds

  console.log("Message 3");
}

// Call the function to see the asynchronous execution
runAsynchronousTasks();
Output will appear here. Click the button to execute the code.
