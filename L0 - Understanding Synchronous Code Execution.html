Synchronous Code Execution

This example demonstrates how JavaScript executes code synchronously, meaning each line of code runs completely before the next one starts. The output will appear in the order the console.log statements are written. 


function runSynchronousTasks() {
  console.log("Task 1");
  console.log("Task 2");
  console.log("Task 3");
}

// Call the function to see the synchronous execution
runSynchronousTasks();
Output will appear in your browser's developer console. Click the button to execute the code.
